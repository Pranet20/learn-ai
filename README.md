# LearnAI - Student Learning Portal

 LearnAI is a fully AI-powered educational platform to help students monitor their learning objectives, create study plans, and find study partners using the power of AI.

## Features

* **Authentication:** Secure API login and register using JWT.

* **Interactive Dashboard:** Displays real-time active course counts, study hour counts, and general proficiency records.

* **Smart Timetable:** Set daily timetables using local storage to send study reminders to your Dashboard

* **Textbook Library:** Access to national standard textbooks and auto-organized study notes.

* **Assessment Engine:** Real-time skill tracking of Math, Physics, Chemistry, and Literature quizzes.

* **StudySync ML Match:** ML algorithm based on your quiz data weakness to match study gaps with a top study partner.

* **Live AI Tutor:** Floating AI assistant using Gemini 2.5 Flash GenAI.

## Tech Stack

* **Frontend:** Next.js, React, Tailwind CSS

* **Backend:** Python, FastAPI, SQLAlchemy, JWT Authentication

* **AI & ML:** Google GenAI SDK (Gemini), Custom Python Weighted Collaborative Filtering
